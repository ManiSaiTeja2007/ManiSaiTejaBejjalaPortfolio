Pathfinding Visualizer (C++)

A desktop app for pathfinding algorithms.

Screenshot of the pathfinding visualizer in action.

Detailed Description

This project is a desktop application built in C++ that visually demonstrates various pathfinding algorithms (e.g., A*, Dijkstra's, BFS, DFS) on a grid. Users can define obstacles, start points, and end points to see how different algorithms navigate the grid to find the shortest or most efficient path.

  • Problem Solved: Provided an interactive and intuitive tool for understanding the mechanics and differences between common pathfinding algorithms, which are fundamental in computer science and game development.
  • My Role: I implemented the core pathfinding algorithms, designed the grid and visualization logic using SFML for graphics rendering, and created the user interface for interaction.
  • Key Features: Implementation of A*, Dijkstra's, Breadth-First Search (BFS), Depth-First Search (DFS), interactive grid manipulation (adding/removing walls), animation of the search process, and path highlighting.
  • Technologies Used: C++, SFML (Simple and Fast Multimedia Library) for graphics and event handling, fundamental data structures (priority queues, sets, etc.) for algorithm implementations.
  • Challenges and Solutions: Optimizing the visualization for smooth animation with large grids was a challenge; this was addressed by drawing only necessary changes per frame and using efficient data structures for algorithm execution. Implementing the algorithms correctly and debugging graph traversals also required careful attention.
  • Learnings: Deepened my understanding of graph theory, algorithm design and analysis, and gained practical experience in C++ desktop application development and game-like rendering using SFML.
